Svarstexten kan innehålla felinformation som följer dokumentationen för <a href="../../../../features-standard/error-reporting/enhanced-error-codes.md">Förbättrade felkoder</a>.
      </td>
   </tr>
   <tr>

      </td>
   </tr>
   <tr>
      <td>500</td>
